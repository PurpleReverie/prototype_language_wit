# Code of Conduct

This project follows the [Contributor Covenant, version 2.1][cc].
The full text is canonical at the link below; please read it before
opening an issue or pull request.

[cc]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Reporting

To report a violation of the Code of Conduct, please email
**tauraj.pro@gmail.com**. All reports will be reviewed and
investigated promptly and fairly.

All community leaders are obligated to respect the privacy and
security of the reporter of any incident.
